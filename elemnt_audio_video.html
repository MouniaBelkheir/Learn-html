In HTML, there is a set of elements used to build the structure of web pages. Of the elements you mentioned, the following are part of the HTML5 specification: 


<header>: Used to define the header of a page or section.
<nav>: Used to define the navigation section.
<main>: Used to define the main content of the page.
<aside>: Used to define sidebar content that may be related to the main content.
<picture>: Used to define a group of image sources with an <img> element.
<figure>: Used to define illustrative content with an optional caption element.
<footer>: Used to define the footer of a page or section.
<figcaption>: Used to define a caption for an image within a <figure> element.
<section>: Used to define a section or part of the content.
<article>: Used to define an article or an independent piece of content.
<ruby>: Used to define ruby annotations, such as pronunciation guides for East Asian text.
